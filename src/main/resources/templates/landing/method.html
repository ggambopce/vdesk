사용방법

PC 사용환경

Windows 또는 Mac PC에서 브이데스크를 사용하는 방법입니다.

1
회원가입 및 로그인

브이데스크 홈페이지에서 회원가입 후 로그인하세요. 구글 계정으로도 간편 로그인이 가능합니다.

2
플랜 구매

대시보드 상단 "구매하기" 버튼을 클릭하여 원하는 플랜을 선택하세요. 일반결제 또는 정기결제로 구독 가능합니다.

3
호스트 프로그램 설치

원격 제어를 당할 PC에 브이데스크 호스트 프로그램을 설치하세요. 설치 후 프로그램을 실행하면 접속키가 표시됩니다.

Windows: VdeskHost_Setup.exe 실행
4
기기 추가 — 접속키 입력

대시보드의 "기기 추가" 버튼을 클릭하고 호스트 프로그램에 표시된 접속키(BX_XXXXXXXXXX)를 입력하세요.

5
원격 연결

등록된 기기 카드에서 더블 클릭하거나 "원격 연결" 버튼을 눌러 즉시 연결하세요. 마이페이지에서 "접속기" 버튼으로도 바로 진입할 수 있습니다.

① 사용방법 버튼

앱 사용환경

스마트폰(Android/iOS)에서 브이데스크 앱으로 PC를 원격 제어하세요.

1
앱 다운로드

Google Play Store 또는 Apple App Store에서 "Vdesk" 앱을 다운로드하세요.

2
로그인

기존 계정으로 로그인하세요. 구글 계정 로그인도 지원합니다.

3
기기 선택 후 연결

등록된 기기 목록에서 연결할 PC를 탭하세요. 자동으로 원격 세션이 시작됩니다.

ONLINE 상태인 기기만 연결 가능합니다
4
터치로 제어

스마트폰 화면을 터치하여 마우스처럼 PC를 제어하세요. 핀치 투 줌으로 화면을 확대/축소할 수 있습니다.

파일전송 방법

원격 PC와 내 기기 사이에서 파일을 쉽게 주고받을 수 있습니다.

1
원격 세션 연결

파일을 전송할 PC와 원격 세션을 먼저 연결하세요.

2
파일 전송 패널 열기

원격 뷰어 상단 도구 모음에서 파일 전송 아이콘을 클릭하세요. 왼쪽 패널(내 PC)과 오른쪽 패널(원격 PC)이 나타납니다.

3
파일 선택 및 전송

전송할 파일을 선택하고 화살표 버튼을 클릭하거나 드래그 앤 드롭으로 파일을 이동하세요.

한 번에 최대 2GB까지 전송 가능

결제 방법

일반결제와 정기결제 중 원하는 방식을 선택하세요.

A
일반결제 (건별 결제)

원하는 기간(1, 3, 6, 9, 12개월)과 PC 대수를 선택하여 한 번에 결제합니다. 카드 결제를 지원합니다.

일반결제 흐름 구매하기 클릭 → 일반결제 탭 → 플랜·수량·기간 선택 → 결제 진행
B
정기결제 (자동 구독)

매월 자동으로 청구되는 구독형 결제입니다. 첫 결제 시 카드 정보를 등록하면 이후 자동으로 청구됩니다.

정기결제 흐름 구매하기 클릭 → 정기결제 탭 → 플랜·수량 선택 → 카드 등록 → 매월 자동 청구
마이페이지에서 언제든지 구독 취소 가능

알아두면 유용한 팁

브이데스크를 더 편리하게 사용하세요.

⌨️
단축키 지원

원격 세션 중 Ctrl+Alt+Del, Win 키 등 모든 단축키가 원격 PC에 전달됩니다.

🖥️
멀티 모니터

원격 PC가 여러 모니터를 사용 중이라면 각 모니터를 개별로 선택하여 볼 수 있습니다.

🔒
세션 잠금

원격 세션 중 실수로 창을 닫아도 세션은 계속 유지됩니다. 대시보드에서 다시 진입하세요.
